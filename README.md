# KET / PET / FCE Reading Test HTML Quiz

Interactive Cambridge reading practice board with:

- PET Reading Tests 1-4 (Parts 1-6)
- FCE Reading Tests 1-4 (Parts 1-3)
- Original source-page scans beside interactive answer controls
- Immediate feedback and saved progress
- Clickable vocabulary with Traditional Chinese meanings
- A per-test vocabulary review list stored in the browser

Open the published site through GitHub Pages or serve this folder with any static web server.
